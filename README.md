# Проект "Фича для IT-отдел крупного банка."

## Описание: Это виджет, который показывает несколько последних успешных банковских операций клиента.

## Установите зависимости:
```
python -m pip install mypy
python -m pip install black
python -m pip install flake8
python -m pip install isort
```

## В проекте используются следующие функции:
* get_mask_card_number - принимает от пользователя номер карты и маскирует его.
```
1111222233334444 -> 1111 22** **** 4444
```
* get_mask_account - принимает от пользователя номер счета и маскирует его.
```
68413314681516381 -> **6381
```
* mask_account_card - что ввел пользователь и передает данные в функции (get_mask_account, get_mask_card_number)
```
Счет 153211637136 -> Счет **7136
Visa 1111 2222 3333 4444 -> Visa 1111 22** **** 4444
```
* get_date - функция, которая принимает дату и время в формате "2024-03-11T02:26:18.671407" и возвращает строку с датой в формате "ДД.ММ.ГГГГ"
```
2024-03-11T02:26:18.671407 -> 11.03.2024
```
* filter_by_state - Функция, которая принимает список словарей и опционально значение для ключа 'state',
    и возвращает новый список словарей.
```
    если (EXECUTED)
    {"id": 41428829, "state": "EXECUTED", "date": "2019-07-03T18:35:29.512364"},
    {"id": 594226727, "state": "CANCELED", "date": "2018-09-12T21:27:25.241689"},
    {"id": 939719570, "state": "EXECUTED", "date": "2018-06-30T02:08:58.425572"},
    {"id": 615064591, "state": "CANCELED", "date": "2018-10-14T08:21:33.419441"},
    ->
    {'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'},
    {'id': 939719570, 'state': 'EXECUTED', 'date': '2018-06-30T02:08:58.425572'}
```
* sort_by_date - Функция принимающая список словарей и необязательный параметр,
    задающий порядок сортировки (по умолчанию — убывание).
    Функция должна возвращать новый список, отсортированный по дате
```
   если (по возрастанию)
    {"id": 41428829, "state": "EXECUTED", "date": "2019-07-03T18:35:29.512364"},
    {"id": 939719570, "state": "EXECUTED", "date": "2018-06-30T02:08:58.425572"},
    {"id": 594226727, "state": "CANCELED", "date": "2018-09-12T21:27:25.241689"},
    {"id": 615064591, "state": "CANCELED", "date": "2018-10-14T08:21:33.419441"},
    ->
    {'id': 939719570, 'state': 'EXECUTED', 'date': '2018-06-30T02:08:58.425572'},
    {'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'},
    {'id': 615064591, 'state': 'CANCELED', 'date': '2018-10-14T08:21:33.419441'},
    {'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}
```

### Для работы функций (filter_by_state, sort_by_date) добавлены специальные типы:
1. Dict
2. List
3. Union
